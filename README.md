# Event Izir App

This app use Node.js, Express and Mongoose.

### Version
1.1.0

### Installation

```sh
$ npm install
```

### Launch app

```sh
$ node app
```

### Test

launch server
```sh
$ npm run test
```
### Build assets

launch server
```sh
$ npm run build
```
